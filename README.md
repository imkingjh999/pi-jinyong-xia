# 🏯 Pi 金庸武侠宠物

一个住在你终端里的金庸武侠角色宠物。

随机化身一位金庸主角，觉醒一门武功绝学，通过日常编程活动修炼武功、升级成长、赚钱买武器。

## 安装

```bash
pi install pi-jinyong-pet
```

## 快速开始

安装后首次启动会随机分配：
- 🎭 **一位金庸主角**（郭靖、令狐冲、萧峰…）
- ⚡ **一门武功绝学**（九阳神功、独孤九剑、降龙十八掌…）

正常使用 pi 编程即可：
- **编码** → 修炼武功、获得经验和金币
- **修炼** → 消耗体力提升技能
- **购买武器** → 花金币在兵器铺选购

## 命令

```
/pet                查看状态
/pet feed           喂食恢复体力（+金币）
/pet train          修炼武功（+金币）
/pet train 内功心法  修炼指定技能
/pet stats          详细数据面板
/pet list           列出所有角色
/pet choose <id>    手动选择角色
/pet reroll         重铸命运（随机新角色+绝学，花费10%金币）
/pet nick <名字>    取昵称

/pet shop           查看武器商店
/pet buy <id>       购买武器
/pet equip <id>     装备武器
/pet weapons        查看我的兵器库

/pet large          放大显示
/pet small          缩小显示
/pet hide           隐藏宠物
/pet show           显示宠物
```

## 金钱系统

- 初始 100 金币
- 编辑文件 +5金、执行命令 +2金、工具调用 +1金、修炼 +3~8金
- 用金币购买武器，或花费金币重铸命运
