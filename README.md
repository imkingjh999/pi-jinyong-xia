|------|------|
| 🎭 **角色** | 25 位金庸经典角色（天龙八部、射雕、神雕、倚天、笑傲、鹿鼎、飞狐） |
| ⚡ **武功** | 32 门武功绝学，五行（金木水火土）相生相克 |

| 🗺️ **江湖** | 227 个金庸世界地点，随机遭遇事件 |
| 🏪 **商铺** | 100 把武器（5 稀有度）+ 100 种道具 |
| 🏛️ **门派** | 10 大门派（少林、武当、峨眉、丐帮、明教、天山、华山、逍遥、桃花岛、日月神教） |
| 💼 **职业** | 8 种江湖职业（剑客、拳师、暗器手、医者、谋士、刺客、铸师、奇门） |

| 📋 **任务** | 100 个江湖任务（5 风险等级），MBTI 性格推导 |

| 🐱 **头像** | Q 版头像支持 Kitty/iTerm2/tmux 图片协议 |
| 📊 **排行榜** | 全服公共排行榜 |

## 📦 安装

```bash
pi install pi-jinyong-xia
```

或手动安装：

```bash
npm install pi-jinyong-xia
```

## 🚀 快速开始

安装后首次启动 pi 会自动随机分配：
- 🎭 **一位金庸主角**（郭靖、令狐冲、萧峰…）
- ⚡ **一门武功绝学**（九阳神功、独孤九剑、降龙十八掌…）

正常使用 pi 编程即可自动获得经验和金币！

## ⚡ 五行武功系统

32 门武功分属五行，**相克倍率 1.5x，被克倍率 0.7x**：

```
金 → 木 → 土 → 水 → 火 → 金
```

| 五行 | 代表武功 |
|------|---------|
| 🤍 金 | 六脉神剑、独孤九剑、弹指神通、金刚不坏体 |
| 💚 木 | 黯然销魂掌、吸星大法、左右互搏、打狗棒法 |
| 💙 水 | 九阴真经、北冥神功、凌波微步、碧海潮生曲 |
| ❤️ 火 | 九阳神功、一阳指、降龙十八掌 |
| 🟤 土 | 乾坤大挪移、太极拳、太极剑法、化骨绵掌 |

最多同时持有 **5 门武功**，每门可修炼至 **20 级**。

| 阶段 | Boss 范例 | 等级区间 |

| 菜鸟期 | 小毛贼、恶霸、木人桩 | 1-10 |

| 传说期 | 独孤求败(幻影)、扫地僧、达摩祖师(幻影) | 55+ |

### 战斗模式

- **⚔️ 自动战斗** — 一键出结果，适合日常刷怪
- **🎮 手动战斗** — 逐回合选择 攻击/武功/防御/道具/逃跑
- **❌ 逃跑** — 不打

### 🎲 骰子事件（每回合自动触发）

| 骰子 | 事件 | 效果 |
|------|------|------|

| 5 | 💥 破防 | 无视Boss 50%防御 |
| 4 | 💚 回春 | 回复8%血量 |
| 3+被克 | 😵 虚弱 | Boss伤害+50% |
| 2 | 💨 闪避 | 伤害降至30% |
| 1 | 🔥 Boss暴走 | Boss攻击×1.5 |

## 🌟 天赋系统

升级获得天赋点，从三条天赋路线中选择，每条路线 5 层进阶：

| 路线 | 方向 | 代表天赋 |
|------|------|---------|
| 💪 战斗之路 | 攻击+暴击 | 蛮力 → 精准 → 剑意通神 → 战神 → 无上剑道 |
| 🛡️ 生存之路 | 防御+血量 | 铁骨 → 坚韧 → 金刚不坏 → 不灭 → 涅槃 |
| 💰 财运之路 | 经验+金币 | 勤勉 → 精明 → 商道 → 富豪 → 天命 |

共 **24 个天赋**，每层需先解锁前一层。

### 🧠 MBTI 性格推导

根据你选择的任务类型和执行方式，自动推导 16 型人格：

- 选择什么任务 → 任务类型的 MBTI 标签
- 怎么执行（计划选项） → 选项的 MBTI 维度映射
- 15 次任务达到 100% 置信度
